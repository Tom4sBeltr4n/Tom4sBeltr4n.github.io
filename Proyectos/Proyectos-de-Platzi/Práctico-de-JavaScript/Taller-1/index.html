Proyecto: Figuras

Figuras es uno de mis algoritmos de JavaScript, cuando ya tenía algunos conocimientos. Aquí manejé JS con algo más de detalle, entendiendo cosas como ámbito de variables y demás. En este se usa un archivo HTML y uno de JS. Fue parte del curso Práctico de JavaScript de Platzi y fue algo motivante. El proyecto toma el lado de un cuadrado (que proporciona el usuario a través de un formulario) y entrega el perímetro o el área del cuadrado, según se requiera. También puede sacar la altura de un triángulo isósceles si se dan 3 de sus lados. No lleva un documento CSS porque no era necesario y quería acabar rápido el proyecto, lo que quizás puede ser un problema porque la apariencia es muy básica, y no es atractivo a la vista. Sin embargo, al no ser un proyecto de desarrollo en sí, sino más bien un trabajo de práctica de JS, no es grave. Aún más; pensé que sin CSS absoluto sería horroroso, entonces añadí un par de líneas bajo una etiqueta <style> y varios inline. También se puede comentar que usé el método de JS   number.toFixed()   para redondear la cifra a dígitos finitos.
