Proyecto: Porcentajes

Este proyecto fue el segundo del curso práctico de JS. Aquí nos pedían hallar el precio de un producto tras un descuento. Para ello, pasamos de hacerlo con variables fijas, luego con una función en la consola, y por último a una interfaz amigable con los usuarios. Se nos pidió también que lo adaptáramos para recibir cupones, pero no lo hice porque no entiendo bien los cupones. Es bastante sencillo, la verdad
